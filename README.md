# MindMate - AI Chat Assistant

MindMate is a modern AI chat assistant built with React and powered by the Gemini API. It provides a clean, intuitive interface for interacting with AI and getting intelligent responses.

## Features

- Clean and modern UI
- Real-time chat with AI
- Message history
- Markdown support
- Responsive design
- Loading animations

## Tech Stack

- React
- Vite
- Axios
- Gemini API
- CSS3

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Suryansh0910/Mindmate.git
cd mindmate
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

## Configuration

The application uses the Gemini API. You can configure the API key in the `src/config/gemini.js` file.
